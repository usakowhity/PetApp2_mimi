# PetApp2_mimi – シャム猫「Mimi」デモ版

PetApp2_mimi は、AIデスクトップペットアプリ「PetApp2 Portable」シリーズの  
**シャム猫「Mimi」デモ版**です。

- プロフィール・画像・動画をあらかじめ登録済み  
- 外部AIツールで生成した素材を `generated/` に同梱  
- ZIP を展開して `run.bat` を実行するだけで動作（Portable版）

---

## 🐱 特徴

- **15種類の状態（n1〜n3, p1〜p12）に対応**
- **AI画像/動画生成ガイド**（Gemini / Copilot / Pika）
- **プロンプト自動生成（日本語・英語）**
- **顔認識・音声認識対応**
- **組み込みPython + ffmpeg 同梱（ZIP版）**

## 🐾 デモ版について

このデモ版には、以下の素材が含まれています：

- シャム猫「Mimi」の画像・動画  
- AI生成素材（`generated/` フォルダ）  
- プロフィール設定済み

ユーザー自身の素材を追加して動作を確認することもできます。

---

## 🌱 アプリのテーマ

本アプリは **「人とペットが心通う、癒しと愛」** をテーマとしています。  
怒り・恐怖・悲しみ・攻撃・虐待などのネガティブな状態は採用していません。
