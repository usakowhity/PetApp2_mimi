# 🐱 PetApp2_mimi

シャム猫「Mimi」の画像・動画を使って楽しめる AI ペットアプリのデモ版です。  
笑顔（顔認識）や声掛け（音声認識）に反応し、15種類の状態に応じて表情や動作が切り替わります。

---

## 🔹 概要

PetApp2_mimi は、PetApp2 Portable をベースにした **猫デモ版（動作確認用）** です。  
ユーザー自身のペットを登録する前に、アプリの動作を確認できます。

---

## 🔹 主な特徴

- シャム猫「Mimi」の画像・動画を同梱  
- 15種類の状態（n1〜n3, p1〜p12）に対応  
- 顔認識・音声認識で状態が切り替わる  
- AI画像/動画生成ガイド（Gemini / Copilot / Pika）に対応  
- インストール不要の Portable 版（run.bat で起動）

---

## 🔹 使い方

1. ZIP を解凍  
2. フォルダ内の **run.bat** を実行  
3. Mimi のプロフィールを確認  
4. PlayWindow で動作開始  
5. 笑顔や声掛けに応じて状態が切り替わります
